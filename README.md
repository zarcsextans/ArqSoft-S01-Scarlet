#  Catálogo de Videojuegos

##  Descripción del Proyecto
Este proyecto consiste en una aplicación web desarrollada con ASP.NET Core MVC que permite administrar un catálogo de videojuegos. El sistema permite visualizar una lista de videojuegos, agregar nuevos títulos y consultar información detallada de cada juego mediante una interfaz moderna con temática gamer.

El objetivo principal del proyecto es aplicar conceptos básicos de desarrollo web utilizando el patrón MVC (Modelo - Vista - Controlador), manejo de vistas Razor y estructuras dinámicas en C#.

---

#  Funcionalidades Principales

-  Visualización de catálogo de videojuegos.
-  Vista detallada de cada videojuego.
-  Agregar nuevos videojuegos mediante formularios.
-  Filtrado por género.
-  Interfaz visual estilo gamer con diseño moderno.
-  Navegación dinámica entre vistas.
-  Uso de tarjetas visuales para mostrar información.

---

#  Tecnologías Utilizadas

- ASP.NET Core MVC
- Razor Views (.cshtml)
- HTML5
- CSS3
- Bootstrap
- Visual Studio
- C#
